FloatButton 悬浮按钮组件

FloatButton 是一个悬浮按钮组件，可用于在页面中提供快捷操作入口，在不同平台（App（vue）、App（nvue）、H5、小程序）上均有良好支持。

基本使用

单一悬浮按钮

使用 :isMenu="false" 可以创建一个单一的悬浮按钮，不展开子菜单。可通过 top 属性调整其位置。

<template>
  <view>
    <up-float-button :isMenu="false" top="90px"></up-float-button>
  </view>
</template>
<script setup>
import { ref } from 'vue';
const list = ref([]);
</script>

带子菜单的悬浮按钮

使用 :isMenu="true" 可展开子菜单，:list 属性绑定子菜单列表，@item-click 监听子菜单点击事件。

<template>
  <view>
    <up-float-button :isMenu="true" top="220px" :list="list" @item-click="handleItemClick"></up-float-button>
  </view>
</template>
<script setup>
import { ref } from 'vue';
const list = ref([
  { key: 'item1', name: 'Item 1', color: '#fff', backgroundColor: 'blue' },
  { key: 'item2', name: 'Item 2', color: '#fff', backgroundColor: 'green' }
]);
const handleItemClick = (item) => {
  console.log('Clicked item:', item);
};
</script>
