    uview-plus Button按钮组件

    uview-plus的Button按钮组件是对uni-app的`button`组件进行二次封装，相较于原生组件，提供了更多`type`主题颜色与`size`尺寸选择。适用于App（vue）、App（nvue）、H5、小程序平台。

    基本使用

    通过`text`属性设置按钮显示的文字，示例：

    <up-button text="月落"></up-button>

    设置按钮的多种形态

    • 样式类型：利用`type`属性设定按钮样式，可选值为`default`（默认）、`primary`、`success`、`info`、`warning`、`error`。
    • 镂空效果：`plain`属性设为`true`时，按钮背景色透明，呈现镂空效果。
    • 细边框：`hairline`属性为`true`时，按钮显示细边框。
    • 禁用状态：`disabled`属性设为`true`，按钮进入禁用状态。
    • 加载图标：`loading`属性为`true`显示加载图标，`loadingText`属性可自定义加载文字提示。
    • 图标显示：`icon`属性设置按钮显示图标，属性值为图标名称。
    • 按钮形状：`shape`属性设为`circle`可使按钮呈圆角形状，默认值为`square`。
    • 渐变颜色：`color`属性支持传入`linear-gradient`渐变色来设置按钮颜色。
    • 按钮大小：`size`属性设置按钮大小，可选值有`large`、`mini`，默认值为`normal`。

    示例代码：

    <template>
    <view style="padding: 20px;">
        <up-button type="primary" text="确定"></up-button>
        <up-button type="primary" :plain="true" text="镂空"></up-button>
        <up-button type="primary" :plain="true" :hairline="true" text="细边"></up-button>
        <up-button type="primary" :disabled="disabled" text="禁用"></up-button>
        <up-button type="primary" loading loadingText="加载中"></up-button>
        <up-button type="primary" icon="map" text="图标按钮"></up-button>
        <up-button type="primary" shape="circle" text="按钮形状"></up-button>
        <up-button text="渐变色按钮" color="linear-gradient(to right, rgb(66, 83, 216), rgb(213, 51, 186))"></up-button>
        <up-button type="primary" size="small" text="大小尺寸"></up-button>
    </view>
</template>
<script setup>
import { ref } from 'vue';
const disabled = ref(true);
</script>

    定义外部样式

    • 非微信小程序平台：可直接给组件添加类名或内联样式修改按钮样式。
    • 微信小程序平台：因编译后页面存在同名元素，直接设置样式可能有问题。推荐使用`custom-style`参数，以对象形式传递样式给组件内部，使用驼峰命名。

    示例代码：

    <template>
    <view style="padding: 20px;">
        <!-- 以下形式在微信小程序会无效，APP和H5有效  -->
        <up-button class="custom-style" text="雪月夜"></up-button>
        <!-- 推荐使用这种方式在微信小程序设置样式  -->
        <up-button :custom-style="{marginTop: '20px', color:'red'}" text="雪月夜"></up-button>
    </view>
</template>
<script setup>
import { ref, reactive } from 'vue';
const disabled = ref(true);
const customStyle = reactive({
    marginTop: '20px',
    color:'red'
});
</script>
<style lang="scss" scoped>
.custom-style {
    color: #ff0000;
    width: 400rpx;
}
</style>

    各家小程序开放能力的对接

    uview-plus已对接uni-app关于`button`组件的所有开放能力（截止2020-04-14），使用方式参考uni-app文档说明。若发现遗漏，请加群反馈。
