uview-plus 组件文档

    uview - plus色彩体系

    uview - plus精心构建了一套全面且实用的调色板，旨在为各类组件提供统一且鲜明的视觉风格，助力打造风格一致的用户界面。该调色板在App（vue）、App（nvue）、H5以及小程序等多个平台上均能实现稳定且一致的色彩呈现。

    使用前提

    uview - plus依赖scss预处理器来优化CSS样式的编写。在使用uview - plus之前，请确保您已在Hbuilder X中安装scss/sass编译插件。若尚未安装，可通过Hbuilder X的“工具” -> “插件安装”路径，搜索并安装该插件。安装完成后，若样式未生效，请尝试重启Hbuilder X。

    颜色分类及使用

    主题色

    uview - plus定义了`primary`（蓝色）、`success`（绿色）、`error`（红色）、`warning`（黄色）、`info`（灰色）作为主题色，这些主题色分别对应不同的视觉感受与业务场景。并且每个主题色都具备`disabled`（禁止）、`dark`（加深）和`light`（变浅）三种状态。

    以`button`组件为例，当`type`参数设置为`primary`时，按钮呈现蓝色；按钮按下时，会应用`primary`的加深颜色（`dark`状态）；当按钮设置为镂空状态（`plain`为`true`）时，背景色变为`primary`的变浅颜色（`light`状态）；而当按钮处于禁止状态时，会使用`primary`的稍浅颜色（`disabled`状态）。

    在全局样式中，通过scss提供了对应的颜色变量名（4.x版本），例如：

    
    // 变量的定义，该部分uview - plus已全局引入，无需您编写
    $up - primary: #3c9cff;
    $up - warning: #f9ae3d;
    $up - success: #5ac725;
    $up - error: #f56c6c;
    $up - info: #909399;
    
    // 在您编写css的地方使用这些变量
    .title {
        color: $up - primary;
        // 其他样式属性
    }

    背景颜色

    uview - plus定义的背景颜色为#f3f4f6，在scss中对应的变量名为`$up - bg - color`，可在编写CSS时调用： 

    
    // 变量的定义，该部分uview - plus已全局引入，无需您编写
    $up - bg - color: #f3f4f6;
    
    // 在您编写css的地方使用这个变量
    .title {
        background - color: $up - bg - color;
    }

    边框颜色

    uview - plus自定义的边框颜色值为#e4e7ed，在scss中对应的变量名为`$up - border - color`，使用示例如下： 

    
    // 变量的定义，该部分uview - plus已全局引入，无需您编写
    $up - border - color: #e4e7ed;
    
    // 在您编写css的地方使用这个变量
    .item {
        border: 1px solid $up - border - color;
    }

    版本差异

    在使用颜色变量时，需留意版本差异。4.x版本的颜色变量无前缀或前缀为`$up -`，而3.x版本前缀为`$u -`。在项目升级或维护过程中，请务必根据实际版本正确调整颜色变量的使用，以确保颜色的正确显示和样式的一致性。
