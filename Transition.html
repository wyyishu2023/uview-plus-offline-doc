    Transition 动画

    该组件用于实现组件的动画过渡效果。

    基本使用

    通过 slot 传入内容，默认使用的是 fade 效果。 

    
    <template>
        <up-transition :show="show">
            <view class="transition"></view>
        </up-transition>
    </template>
    <script setup>
    import { ref } from 'vue';
    
    const show = ref(true);
    </script>

    动画模式

    通过 mode 传入效果模式，目前支持：

    • fade：淡入
    • fade-up：上滑淡入
    • fade-down：下滑淡入
    • fade-left：左滑淡入
    • fade-right：右滑淡入
    • slide-up：上滑进入
    • slide-down：下滑进入
    • slide-left：左滑进入
    • slide-right：右滑进入
    • zoom-in：缩放
    • zoom-out：缩放
    
    <template>
        <up-transition :show="show" mode="zoom-in">
            <view class="transition"></view>
        </up-transition>
    </template>
    <script setup>
    import { ref } from 'vue';
    
    const show = ref(true);
    </script>
